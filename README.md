## Objetivos:
- Criar um projeto Ruby on Rails MVC
- Criar um blog no seguindo o [template](https://getbootstrap.com/docs/5.0/examples/blog/)
- CRUD de artigos e categorias
- Comentários
- Autenticação com Devise
- Autorização com Pundit
- Testes com RSpec
- Bootstrap
- l18n, Deploy, CI, etc...
